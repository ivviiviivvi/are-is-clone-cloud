# Rclone Guides

This directory contains practical guides and tutorials for using rclone
in various scenarios.

## Available Guides

### Personal Knowledge Base with Multiple Cloud Drives

**File:** [personal-knowledge-base.md](personal-knowledge-base.md)

A comprehensive guide showing how to build a personal knowledge base
using multiple cloud storage providers (iCloud Drive, Dropbox, Google
Drive, and OneDrive). This guide covers:

- Setting up and configuring multiple cloud drive remotes
- Creating unified views using union and combine backends
- Syncing strategies for redundancy and organization
- Mounting cloud storage locally
- Automation scripts for scheduled syncs
- Encryption for sensitive data
- Monitoring and maintenance

**Supporting Files:**

- `rclone.conf.example` - Example configuration file showing all
  remote setups
- `sync-knowledge-base.sh` - Bash script for Linux/Mac automation
- `sync-knowledge-base.ps1` - PowerShell script for Windows automation

## Using These Guides

1. Read the main guide document for your use case
2. Follow the step-by-step instructions
3. Use the example configuration files as templates
4. Customize the automation scripts for your needs
