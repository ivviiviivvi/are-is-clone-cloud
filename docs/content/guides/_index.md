This section contains practical guides and tutorials for using rclone

## Available Guides

### [Personal Knowledge Base with Multiple Cloud Drives](/guides/personal-knowledge-base/)

A comprehensive guide showing how to build a personal knowledge base
using multiple cloud storage providers (iCloud Drive, Dropbox, Google
Drive, and OneDrive). Learn how to:

- Configure multiple cloud drive remotes
- Create unified views using union and combine backends
- Implement syncing strategies for redundancy and organization
- Mount cloud storage locally for easy access
- Automate syncs with provided scripts
- Encrypt sensitive data
- Monitor and maintain your knowledge base

This guide includes example configuration files and automation scripts
for both Linux/Mac and Windows.
